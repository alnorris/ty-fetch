# ty-fetch playground

Try ty-fetch with fully typed API calls.

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/alnorris/ty-fetch?quickstart=1)

Open `playground/example.ts` and explore — types work immediately in Codespaces.

## Or run locally

```bash
git clone https://github.com/alnorris/ty-fetch.git
cd ty-fetch/playground
npm install
```

Open `example.ts` in VS Code:
- Hover over API calls to see typed responses
- Try changing URLs to see path validation
- Use autocomplete inside the URL strings

> **VS Code:** Make sure you're using the workspace TypeScript version.
> Command Palette → TypeScript: Select TypeScript Version → Use Workspace Version
