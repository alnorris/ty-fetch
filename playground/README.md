# ty-fetch playground

Try ty-fetch in your editor — fully typed API calls with zero codegen.

## Quick start

```bash
npm install
```

Open `example.ts` in VS Code and explore:
- Hover over API calls to see typed responses
- Try changing URLs to see path validation
- Use autocomplete inside the URL strings

> **VS Code:** Make sure you're using the workspace TypeScript version.
> Command Palette → TypeScript: Select TypeScript Version → Use Workspace Version
